How does it work?

A tiny neural network learns to memorize a single image — storing everything needed to repaint it, pixel by pixel, from scratch.

Embeddings A low-resolution grid of learned tiles — one per region. Each tile encodes what this area looks like: a blurry hint the network looks up by position. A grid of learned region hints.
Neurons Three small layers of arithmetic that read the tiles and learn how to paint them into real R·G·B pixels. Each layer refines the result a little more. Layers that paint pixels from hints.
Training
  • Construct and compare.
  • Score the error.
  • Nudge numbers.
  • Repeat thousands of times.
The image gradually sharpens.
Fun things to try
  • Pause training and click a single embedding tile — see its isolated influence on the image.
  • Switch the activation to None and watch the network struggle without non-linearities.
  • Paint a region with the ROI Mask to tell the network what to preserve in priority.
  • Hit Shake occasionally — it jolts the model out of a local minimum.

◈ NNC — Neural Image Compressor
Trains an implicit neural representation (embedding grid + 3-layer MLP) on a single image entirely in WebGPU compute shaders. The activation function is selectable (SIREN, Tanh, Softsign, None). The trained model can be exported as a compact GLSL fragment shader for platforms like ShaderToy.
URL Parameters
Append any of these to auto-configure and immediately start training (skips loading the default model):
grid=NEmbedding grid resolution — 16, 32, or 64
EMB=NEmbedding channels per cell — 4, 8, or 16
MLP1=NLayer 1 output width — 4, 8, 16, 32, or 64
MLP2=NLayer 2 output width — 4, 8, 16, 32, or 64
iters=NMax training steps (0 = run forever)
embedLr=NEmbedding learning rate slider value (0–100)
mlpLr=NMLP learning rate slider value (0–100)
smoothEnable smooth interpolation
nosmoothDisable smooth interpolation
8qatEnable 8-bit quantization-aware training for the MLP
4bPack embeddings as 4-bit (requires EMB ≥ 8)
8bPack embeddings as 8-bit (default)
noneDisable MLP quantization (fp32 weights)
numLoops=NNumber of times each EMB/MLP phase is repeated per super-cycle (1–8)
Example: ?grid=64&EMB=16&MLP1=16&MLP2=8&8qat&4b&iters=4000&numLoops=3
Model Settings
Embedding GridSpatial resolution of the feature grid (N×N). Larger = finer detail, bigger model.
Embedding ChannelsFeature dimensions per grid cell. More channels = richer representation.
MLP Width 1Output width of Layer 1 (embCh → W1). Larger = more capacity, slower training.
MLP Width 2Output width of Layer 2 (W1 → W2). Independent from W1.
ActivationHidden-layer activation function. SIREN (sin) works well for natural images. Tanh and Softsign are smooth alternatives. None (identity) is useful for educational purposes — makes the MLP a purely linear model.
QuantizationMLP weight precision. QAT trains with simulated 8-bit quantization, yielding a much smaller GLSL export.
Emb BitsQuantization depth for the embedding grid. 4-bit halves storage vs 8-bit.
Smooth InterpBicubic-like grid sampling via IQ's smooth-step trick. Improves quality at negligible cost.
No OffsetDisable per-plane UV jitter. Each embedding plane normally samples at a small random offset to increase effective resolution.
Training Settings
Max IterationsTraining stops after N steps. Set to 0 to run indefinitely.
Embeddings LRAdam learning rate for the embedding grid (log scale: 1e-4 … ~1.2e-1; default 3.0e-2).
MLP LRAdam learning rate for MLP weights (log scale: 1e-4 … ~1.2e-1; default 3.0e-2).
LossMSE over RGBA with ITU-R BT.601 luminance weights on R/G/B (0.299 / 0.587 / 0.114). Alpha is weighted equally and supports transparent images.
Emb / MLP ratioHow many embedding-only Adam steps are taken per MLP step. Higher = focus training on the grid.
Num LoopsRepeats each phase N times per super-cycle. With ratio=5 and loops=3: 15 consecutive EMB steps then 3 MLP steps, then repeat.
Bwd pixel strideSubsample every Nth pixel during backprop. Speeds up steps at the cost of noisier gradients.
Visualization
Three panels update live during training:
SourceInput image. Click or drop a new image to replace it. Paint the ROI mask by dragging.
EmbeddingsAll embedding channels shown as grayscale thumbnails. Click a thumbnail to flip all other channels in the forward pass, leaving the clicked one unchanged.
LayersInference flow diagram: embedding → L1 → L2 → L3 → output. Pre-activation values are shown per-neuron with per-layer min/max normalisation. Updates every few steps.
OutputDecoded result at full resolution. Use the zoom slider to upscale.
ROI Mask
Paint focus regions directly on the source image. The training loss is scaled by ROI Strength inside painted areas, biasing the network to prioritize those regions. The mask decays over time unless frozen. Auto Mask detects high-variance (high-detail) areas automatically.
Brush RadiusSize of the paint brush in pixels.
ROI StrengthLoss multiplier inside painted regions. Higher = stronger bias toward those areas.
Freeze MaskPrevent the mask from decaying over training steps.
